
## 项目结构参考
### 所有php项目都遵循此结构
**app文件是**

```
fbi/
├── app/
│   ├── controllers/API/          # API 控制器
│   ├── BLL/                       # 业务逻辑层
│   ├── config/                    # 配置信息
│   ├── common/functions.php        # 自定义函数
│   ├── models/                    # 数据模型
│   ├── models/bicenter/           # bicenter 数据库模型
│   ├── modules/
│   │   ├── Ph/                    # 菲律宾模块
│   │   └── Th/                    # 泰国模块
│   └── library/                   # 工具类
```
