## 根因分析（Root Cause）

### 问题清单

| 问题 | 根因 | 预防措施 |
|------|------|----------|
| 测试脚本位置不规范 | 没有事先确认项目测试目录结构 | 建立标准测试目录 `tests/Unit/{需求号}/` |
| 复制代码而非调用实际方法 | 对测试目的理解偏差 | 测试必须调用实际方法，验证方法行为 |
| Phalcon 初始化困难 | 框架依赖大量服务和配置 | 编写标准 bootstrap.php 模板 |
| 数据库指向错误环境 | 未确认国家配置文件 | 从 `.env.{country}` 加载配置 |
| CLI 模式缺少 Web 变量 | 部分代码依赖 `$_SERVER` | bootstrap 中预设环境变量 |
