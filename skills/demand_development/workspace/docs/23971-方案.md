## 二、方案概述

### 背景
每月 6 号，针对上月"客户投诉-派件投诉-虚假妥投"（punish_category=20）的处罚记录，
按照剔除规则过滤后，汇总每名员工的警告信计次数据，写入新表。

### 核心数据流

```
abnormal_message (punish_category=20)
    ↓ 过滤：isappeal 判断（1=计次 / 5且isdel=1=不计次）
    ↓ 剔除：Buyer Return 客户ID
    ↓ 剔除：爆仓网点（AbnormalWhiteListBLL->getDayList, type=2）
    ↓ 剔除：hire_type IN (2,3,4,13)
    ↓ 剔除：CS回访结果=客户已收到
    ↓ 剔除：跨网点支援工号（hr_staff_apply_support_store）
    ↓ 剔除：CDC（暂留空）
    ↓
staff_warning_letter_count（db_bi_center 新表）
```

## 六、BLL 核心逻辑

### 6.1 入口方法

```php
public function run(string $statMonth): void
{
    // 1. 计算上月时间范围
    $startDate = $statMonth . '-01';
    $endDate   = date('Y-m-d', strtotime($startDate . ' +1 month'));

    // 2. 拉取需计次的处罚记录（isappeal=1 或 保持原判但非申诉成功）
    $records = $this->getCountableRecords($startDate, $endDate);

    // 3. 批量应用剔除规则
    $records = $this->applyExclusionRules($records, $startDate, $endDate);

    // 4. 按员工汇总
    $summary = $this->aggregateByStaff($records, $statMonth);

    // 5. 写入 staff_warning_letter_count（先删后写，保证幂等）
    $this->upsertResult($summary, $statMonth);
}
```

### 6.2 数据查询

```sql
-- 查询条件：
-- punish_category = 20（虚假妥投）
-- isappeal IN (1, 2, 3, 4)  即：排除 isappeal=5(申诉成功 isdel=1)
-- isdel = 0
-- abnormal_time 在上月范围内

SELECT
    am.id,
    am.pno,
    am.punish_category,
    am.abnormal_time,
    am.staff_info_id,
    am.store_id,
    am.extra_info
FROM abnormal_message am
WHERE am.punish_category = 20
  AND am.isdel = 0
  AND NOT (am.isappeal = 5 AND am.isdel = 1)  -- 排除申诉成功
  AND am.abnormal_time >= '{startDate}'
  AND am.abnormal_time  < '{endDate}'
```

> 注：`isappeal=5` 时 `isdel` 会被改为 1，所以此条件等价于「申诉成功」

### 6.3 剔除规则

#### 规则1：Buyer Return 客户ID

```php
private const BUYER_RETURN_CLIENT_IDS = [
    'AA0649', 'AA0650',                                           // shein
    'AA0606', 'AA0657', 'AA0707', 'AA0892', 'AA0461', 'AA0868', 'AA0867', // 其他平台

// 从 extra_info->client_id 判断
private function isExcludedByBuyerReturn(array $record): bool
{
    $extra = json_decode($record['extra_info'], true);
    $clientId = $extra['client_id'] ?? '';
    return in_array($clientId, self::BUYER_RETURN_CLIENT_IDS);
}
```

#### 规则2：爆仓网点

```php
// 每天缓存一次，key: date
private function getBurstStoreIds(string $date): array
{
    $list = (new AbnormalWhiteListBLL())->getDayList($date, 2, 'store_id');
    return array_column($list, 'store_id');
}

// 每条记录按 abnormal_time 判断
private function isExcludedByBurstStore(array $record, array &$burstStoreCache): bool
{
    $date = substr($record['abnormal_time'], 0, 10);
    if (!isset($burstStoreCache[$date])) {
        $burstStoreCache[$date] = array_flip($this->getBurstStoreIds($date));
    }
    return isset($burstStoreCache[$date][$record['store_id']]);
}
```

#### 规则3：员工雇佣类型

```php
// 批量查询 hire_type，过滤 IN (2,3,4,13)
```

### 6.4 汇总写入

```php
// 按 staff_info_id 汇总
private function aggregateByStaff(array $records, string $statMonth): array
{
    $result = [];
    foreach ($records as $record) {
        $staffId = $record['staff_info_id'];
        if (!isset($result[$staffId])) {
            $result[$staffId] = [
                'stat_month'    => $statMonth,
                'staff_info_id' => $staffId,
                'store_id'      => $record['store_id'],
                'warning_count' => 0,
                'detail'        => [],
            ];
        }
        $result[$staffId]['warning_count']++;
        $result[$staffId]['detail'][] = [
            'abnormal_message_id' => $record['id'],
            'pno'                 => $record['pno'],
            'punish_category'     => $record['punish_category'],
            'abnormal_time'       => $record['abnormal_time'],
        ];
    }
    return $result;
}

// 幂等写入：先删当月数据，再 INSERT
private function upsertResult(array $summary, string $statMonth): void
{
    $db = $this->getDI()->get(EtlDbEnums::DB_BI_CENTER);
    $db->execute("DELETE FROM staff_warning_letter_count WHERE stat_month = '{$statMonth}'");
    $now = custom_gmdate(time());
    foreach (array_chunk($summary, 500) as $chunk) {
        // batch insert
        $this->batchInsert('staff_warning_letter_count', array_map(fn($r) => [
            ...$r,
            'detail'     => json_encode($r['detail']),
            'created_at' => $now,
            'updated_at' => $now,
        ], $chunk), EtlDbEnums::DB_BI_CENTER);
    }
}
```
