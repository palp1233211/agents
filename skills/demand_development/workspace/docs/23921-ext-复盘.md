# 复盘文档 — 23921-ext 客户尺寸/重量字段为0时差值计算优化

> 复盘时间：2026-04-09

---

## 目标（Goal）

当客户尺寸/重量字段为 0 时，差值计算返回 0 而非不合理的负值。

---

## 结果（Result）

### 实际交付

| 内容 | 状态 |
|------|------|
| weight_diff 计算（client_weight=0 返回 0） | ✅ |
| size_diff 计算（任一为 0 返回 0） | ✅ |
| length/width/height_diff 计算（client=0 返回 0） | ✅ |

### 偏差说明

无偏差，按方案完成。

---

## 根因分析（Root Cause）

### 问题背景

客户字段可能为 0，原计算逻辑 `flash - client` 会得到不合理的负差值。

### 解决方案

- 单独判断：每个差值独立检查 client 字段是否为 0
- size_diff 特殊处理：任一尺寸为 0 则返回 0

---

## 可复用模式（Reusable Patterns）

### 检查项

1. **差值计算需考虑边界值**：client 字段可能为 0
2. **测试用例覆盖边界**：必须包含 client=0 的测试场景

### 代码模式

```php
// 单字段差值
$diff = $client == 0 ? 0 : bcsub($flash, $client, 2);

// 组合字段差值（任一为 0 则返回 0）
$diff = ($client_a == 0 || $client_b == 0 || $client_c == 0) ? 0 : bcsub(...);
```
