# 开发方案 — 23921-ext 客户尺寸/重量字段为0时差值计算优化

> 创建时间：2026-04-09
> 作者：lwc
> 涉及项目：bi-common

---

## 一、需求背景

客户尺寸/重量字段可能为 0，此时计算差值会得到不合理的负值，需要优化差值计算逻辑。

---

## 二、接口变更（bi-common）

### 2.1 getListDetailsProcess() 方法

**文件：** `app/BLL/ParcelReweighAppealReviewBLL.php`

**位置：** 第 556-557 行

**变更前：**
```php
'weight_diff' => bcsub($flash_parcel_weight, $client_parcel_weight, 2),
'size_diff' => bcsub(($flash_parcel_length + $flash_parcel_width + $flash_parcel_height), ($client_parcel_length + $client_parcel_width + $client_parcel_height), 2),
```

**变更后：**
```php
'weight_diff' => $client_parcel_weight == 0 ? 0 : bcsub($flash_parcel_weight, $client_parcel_weight, 2),
'size_diff' => ($client_parcel_length == 0 || $client_parcel_width == 0 || $client_parcel_height == 0) ? 0 : bcsub(($flash_parcel_length + $flash_parcel_width + $flash_parcel_height), ($client_parcel_length + $client_parcel_width + $client_parcel_height), 2),
```

---

### 2.2 getListAdjustmentExcel() 方法

**文件：** `app/BLL/ParcelReweighAppealReviewBLL.php`

**位置：** 第 240-242 行

**变更前：**
```php
$data[$k]['length_diff'] = bcsub($datum['flash_parcel_length'], $datum['client_parcel_length'], 2);
$data[$k]['width_diff'] = bcsub($datum['flash_parcel_width'], $datum['client_parcel_width'], 2);
$data[$k]['height_diff'] = bcsub($datum['flash_parcel_height'], $datum['client_parcel_height'], 2);
```

**变更后：**
```php
$data[$k]['length_diff'] = $datum['client_parcel_length'] == 0 ? 0 : bcsub($datum['flash_parcel_length'], $datum['client_parcel_length'], 2);
$data[$k]['width_diff'] = $datum['client_parcel_width'] == 0 ? 0 : bcsub($datum['flash_parcel_width'], $datum['client_parcel_width'], 2);
$data[$k]['height_diff'] = $datum['client_parcel_height'] == 0 ? 0 : bcsub($datum['flash_parcel_height'], $datum['client_parcel_height'], 2);
```
